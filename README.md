# Community Event Portal

A full-stack web application developed as part of academic coursework to demonstrate Front-End Development, Database Management, and Core Java concepts. This project provides a platform for users to discover, register, and participate in community events through an interactive and responsive web interface.

---

## Project Overview

The Community Event Portal enables users to:

* View upcoming community events
* Browse event galleries
* Register for events online
* Find nearby events using geolocation
* Access FAQs and support information
* Store user preferences using Local Storage
* Interact with a responsive and user-friendly interface

## How to Run

### Front-End

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` using any modern web browser.

### Database

1. Open MySQL Workbench or phpMyAdmin.
2. Import the `queries.sql` file.
3. Execute the queries.

### Java Programs

1. Open the Java files in VS Code or Eclipse.
2. Compile the program:

  -> javac FileName.java

3. Run the program:

  -> java FileName
